Экран приложения состоит из 4 основных компонентов:
- Строка меню
- Строка поиска
- Строка управления таблицей
- Окно таблицы

Строка меню имеет следующую структуру:
- Файл:
- - Новый: создает новый файл базы данных в выбранном вами месте и открывает его для редактирования.
- - Открыть: открывает для редактирования ранее созданный файл базы данных по вашему выбору.
- - Создать резервную копию: создает резервную копию текущей базы данных в определенной системной папке.
- - Восстановить из резервной копии: позволяет вам выбрать файл резервной копии и восстановить из него базу данных. Если никакая база данных не открыта в данный момент, вам будет предложено выбрать место для сохранения новой базы данных.
- - Выход: закрывает программу.
- Выделение:
- - Выделить все: устанавливает галочку напротив каждой записи в таблице.
- - Обратить выделение: устанавливает галочку там, где ее нет, и убирает галочку там, где она есть.
- - Снять выделение: убирает все установленные галочки.
- Справка:
- - Помощь: отображает этот текст.
- - О программе: отображает данные о программе и ее создателе.

Строка поиска состоит из трех элементов:
- Селектор режима поиска: выбирает столбец, по которому будет осуществлен поиск.
- Строка поиска: в нее вводится поисковый запрос. Нечеткий поиск не поддерживается. В случае, если поисковый запрос имеет неверный формат, это будет отображено в строке поиска.
- Кнопка "Найти": осуществляет поиск по введенному запросу, или возвращает исходную таблицу если строка поиска пуста.

Строка управления таблицей позволяет осуществлять различные действия с данными в таблице.
Строка управления таблицей состоит из текстовых полей и одного выпадающего меню, а также кнопки действия.
Кнопка действия имеет три возможных состояния:
- "Добавить": состояние по умолчанию. Активно, если ни одна запись в таблице не отмечена галочкой. При нажатии на кнопку в базу данных добавляется новая запись, соответствующая данным, находящимся в текстовых полях и выпадающем меню.
- "Удалить": активно, если хотя бы одна запись отмечена галочкой и текстовые поля и выпадающее меню пустые. При нажатии на кнопку из базы данных будут удалены записи, отмеченные галочкой.
- "Изменить": активно, если хотя бы одна запись отмечена галочкой и хотя бы одно текстовое поле и/или выпадающее меню содержат в себе какое-либо значение, кроме поля "Номер". При нажатии на кнопку все отмеченные записи в базе данных будут отредактированы. Поля, оставленные пустыми, отредактированы не будут. Поле "Номер" можно изменить только для одной записи и только при условии что оно не совпадает с номером другой записи.

Окно таблицы содержит текстовое табличное представление записей в базе данных. Напротив каждой записи с правой стороны есть чекбокс для выбора строки путем установки в него галочки.
